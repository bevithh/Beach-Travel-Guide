Enhancements to Beach Tours Website

Below are the enhancements I've made to my website about beach tours in Melbourne, Australia, along with descriptions and implementation details.

1. Interactive Image Map for Tour Locations

This enhancement improves user engagement by providing an interactive image map where users can click on different beach locations for tour information.

To implement this, I used an HTML `` element along with `` tags to define clickable areas on an image. Here's a simplified example: 

        
        <img src="map.jpg" usemap="#beachMap" alt="Beach Map">
        <map name="beachMap">
            <area shape="circle" coords="100,100,50" href="beach1.html" alt="Beach 1">
            <area shape="circle" coords="200,200,50" href="beach2.html" alt="Beach 2">
            
        </map>

Source: Self-implemented technique using HTML image maps.

Explore the interactive map on the homepage.

2. Custom CSS Grid Layout for Tour Packages

This enhancement enhances the visual appeal of tour package listings by using a custom CSS grid layout with hover effects.

I implemented this using CSS3 grid properties combined with hover effects. Here's a snippet of the CSS code: 

        
        .tour-packages {
            display: grid;
            grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
            grid-gap: 20px;
        }

        .tour-package:hover {
            transform: scale(1.05);
            box-shadow: 0 0 10px rgba(0, 0, 0, 0.3);
        }

Source: Self-implemented CSS grid layout and hover effects.

View the custom grid layout on the homepage.
